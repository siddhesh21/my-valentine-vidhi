# My Valentine — Vidhi

A small, heartfelt Valentine website created for Vidhi.

---

## Preview

- Open [index.html](index.html) in your browser for a local preview.

```bash
python3 -m http.server 8000
```

## What this is

This repository contains a simple static site built as a personal Valentine surprise. It includes the main page, styles, and assets to create a sweet, responsive mini-site.

## Files

- [index.html](index.html) — Main page
- [styles.css](style.css) — Styling for the site
- README.md — This file

## Customization

- To change the displayed name or message, edit [index.html](index.html).
- To tweak colors, fonts, or layout, edit [styles.css](style.css).

## Hosting

- You can host this on GitHub Pages by pushing the repository to GitHub and enabling Pages in the repository settings.
